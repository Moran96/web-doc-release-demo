    蓝牙配网插件

    抽离自原插件 quecBle

    快速接入

    1. API使用方式的变更

    首先注意以下几个API使用方式的变更;

    其中1、2未被使用；3被使用了一次，但不涉及代码更改

    // 采用 NPM 方式导入模块（或使用 CDN 下载js文件到本地，直接引用 js 文件）
    import quecBle from '@quec-wx-mp/plugin-ble'

    // 1. 配置 BleGuideUrl （该步骤可省略，默认使用 https://aiot-bootstrap.quectel.com）
    quecBle.setBleGuideUrl('https://aaa.bbb.com')

    // 2. 配置 getMccAddr 接口请求超时时间（该步骤可省略，默认使用180000）
    quecBle.setOutTime(5000)

    // 3. 直接使用对应方法
    quecBle.getMccAddr({...})
