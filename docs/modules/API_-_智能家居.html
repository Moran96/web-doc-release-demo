    模块 API - 智能家居

    api-smart-home

    抽离自原插件 quecHouse

    快速接入

    1. 引入插件

    // 采用 NPM 方式导入模块（或使用 CDN 下载js文件到本地，直接 import js 文件）
    import smartHome from '@quec-wx-mp/api-smart-home'
    const plugin = requirePlugin("quecPlugin")

    plugin.use(smartHome)

    2. 修改代码

    将原插件中 plugin.quecHouse.xxx 替换为 plugin.smartHome.xxx

    迭代需知

    1. 蓝牙配网V2组件引用了当前模块，需要在BLE V2迁移到example后做适配

    2. quecHouse.getFamilyList、quecHouse.getFamilyModeConfig 的处理：

    • 新增 quecCore 模块，将 jsUtil 中使用的quecHouse相关方法（getFid/getMode）迁移到 quecCore
    • Example 中对 getFid/getMode 的引用，需要改为基于 quecCore
    • getFamilyList/getFamilyModeConfig 定义了两处（quecCore和 api-smart-home）, 后续迭代要分别处理
