    函数 deleteHistoryV2

    • deleteHistoryV2(
          options: {
              roleIds: number[];
              deviceInfoId?: number;
              productKey?: string;
              deviceKey?: string;
              endUserId?: string;
              success?: (res: ResponseData) => void;
              fail?: (res: Error) => void;
              complete?: () => void;
          },
      ): void

      聊天记录删除

      参数

      • options: {
            roleIds: number[];
            deviceInfoId?: number;
            productKey?: string;
            deviceKey?: string;
            endUserId?: string;
            success?: (res: ResponseData) => void;
            fail?: (res: Error) => void;
            complete?: () => void;
        }
        • roleIds: number[]

          角色ID列表

        • 可选deviceInfoId?: number

          设备信息ID（优先以deviceInfoId为主，没有的话处理pk、dk、endUserId）

        • 可选productKey?: string

          产品PK

        • 可选deviceKey?: string

          设备DK

        • 可选endUserId?: string

          终端用户ID

        • 可选success?: (res: ResponseData) => void

          接口调用成功的回调函数

        • 可选fail?: (res: Error) => void

          接口调用失败的回调函数

        • 可选complete?: () => void

          接口调用结束的回调函数（调用成功、失败都会执行）

      返回 void

      示例

      const plugin = requirePlugin('quecPlugin')
      plugin.ai.deleteHistoryV2({
        roleIds: [1],
        success (res) {},
        fail (res) {},
        complete (res) { }
      })
