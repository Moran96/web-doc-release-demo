      获取基础信息

      • options: {
            complete?: () => void;

            fail?: (res: Error) => void;

            success?: (res: ResponseData) => void;
        }
        • 可选complete?: () => void

          获取宠物信息结束

        • deviceKey: string

          设备 deviceKey

        • 可选fail?: (res: Error) => void

          获取宠物信息失败

        • productKey: string

          产品 productKey

        • 可选success?: (res: ResponseData) => void

          获取宠物信息成功

      返回 void

      示例

      const plugin = requirePlugin('quecPlugin')
      plugin.info.getInfo({
          productKey: '',
          deviceKey: '',
          success (res) {},
          fail (res) {},
          complete (res) { }
      })
